Mis Proyectos

Algunos de mis proyectos universitarios.

Colibri 2D

Este proyecto utiliza OpenGL para crear una representación en 2D de un colibrí con un estilo de baja poli. A través de técnicas de renderizado gráfico, el diseño resalta por su simplicidad geométrica y combinación de colores. Es un ejemplo de cómo OpenGL puede ser usado para generar gráficos complejos y estéticamente agradables en aplicaciones visuales.

SocialNETx

Desarrollada en Java con una base de datos local en SQL Server, esta red social permite a los usuarios registrarse, crear perfiles y compartir contenido. SocialNETx está diseñada para ofrecer una experiencia de usuario fluida y segura, con almacenamiento local de datos para un acceso rápido y eficiente.

Alma fria APP

Aplicación móvil desarrollada en .NET MAUI para la paletería Alma Fría. La app permite a los usuarios explorar una variedad de productos de heladería, agregar sus favoritos al carrito de compras y realizar pedidos.
